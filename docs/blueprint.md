# **App Name**: IOTA Stay

## Core Features:

- NFT Minting: Enable hotel owners to mint NFTs representing room availability, including details like date, room type, and price. This is accomplished using IOTA.
- Room Booking: Allow visitors to book rooms by trading the room availability NFT from the hotel owner. This guarantees unique bookings. This is accomplished using IOTA.
- Connect Wallet: Display a connect wallet page when the IOTA wallet is not connected.
- Differentiated Interface: Implement role based screens, so that different features will be displayed to hotel owners versus visitors.
- Room Image Updates: Allows hotel owners to update the image and hash of a listed room. The LLM is used as a tool to suggest new promotional images, based on analyzing current booking data and market trends.
- Browse NFT Listings: List NFTs on the application. It will support filtering to assist the user in finding desired NFTs.
- Get NFT Details: NFT details pages display associated NFT details on the app, such as name, image and booking date.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke trust and dependability.
- Background color: Very light grayish-blue (#F0F4F9), almost white, creating a clean, spacious feel.
- Accent color: Vivid orange (#FF9800) for call-to-action buttons and interactive elements.
- Body font: 'Inter', sans-serif, for a modern, neutral, highly readable appearance.
- Headline font: 'Space Grotesk', sans-serif, for headlines, paired with Inter for body text, lends a techy, scientific touch to attract attention
- Use minimalist line icons to represent room types, amenities, and booking actions.
- Emphasize a clean, card-based layout to present hotel and booking information clearly and concisely.
- Implement subtle transitions and animations when NFTs are traded or booking statuses change.
